A simple Streamlit app that shows how to build a chatbot using OpenAI's GPT-3.5.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://chatbot-template.streamlit.app/)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```

3. How It Works
User enters an OpenAI API key
User types messages in the chat input
Messages are stored in session state
The app sends the conversation history to OpenAI's API
Responses stream back and are displayed in real time
The conversation persists during the session
